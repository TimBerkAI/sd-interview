# Architecture Interview Platform

Backend service for conducting system design interviews.
Built with **FastAPI** · **PostgreSQL 17** · **SQLAdmin** · **Alembic** · **uv**

---

## Stack

| Layer       | Technology              |
| ----------- | ----------------------- |
| Runtime     | Python 3.12             |
| Framework   | FastAPI 0.135           |
| ORM         | SQLAlchemy 2.0 (async)  |
| Migrations  | Alembic                 |
| Admin       | SQLAdmin                |
| Database    | PostgreSQL 17           |
| Proxy       | Nginx                   |
| Package mgr | uv                      |
| Container   | Docker / Docker Compose |
